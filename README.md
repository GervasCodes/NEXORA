# NEXORA

## Overview

NEXORA is a regional multi-vendor e-commerce platform designed to connect buyers, sellers, and delivery service providers within a unified digital marketplace.

The platform enables secure online shopping, seller management, order processing, integrated delivery tracking, and customer communication.

## Technology Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MySQL

### Authentication

* JSON Web Token (JWT)

### Media Storage

* Cloudinary

### Email Service

* Nodemailer

## Development Status

Project initialization and Software Requirements Specification (SRS) phase.
